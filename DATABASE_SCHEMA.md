# Database Schema Reference

Two schema levels exist in the same PostgreSQL instance:

- **PCI Master** ‚Äî `public` schema, system-wide data (tenants, users, roles, master data)
- **Tenant Schema** ‚Äî `tenant_{uuid}` per council, isolated business data (applications, documents, payments)

---
